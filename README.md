# 🏠 Bangalore House Price Prediction
## 📸 App Preview

![App Screenshot](./assets/app_screenshot.png)

A Machine Learning web app that predicts house prices in Bangalore based on user inputs like square feet, BHK, location, bathrooms, and balcony.

---

## 🚀 Live Demo

👉 [Click Here to Use the App](https://benguluruhousepriceprediction-ig5fjpfbhnwtzxhlsbivbv.streamlit.app/)

---


## 📌 Features

- 📊 Predict house price instantly
- 🏙️ Location-based prediction
- 🛏️ Supports BHK, Bathrooms, Balcony inputs
- ⚡ Fast and interactive UI using Streamlit
- 🧠 ML model trained on real dataset

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
- Scikit-learn 🤖
- Pandas & NumPy 📊
- Joblib (Model Saving)
